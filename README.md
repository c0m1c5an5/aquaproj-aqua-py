# aquaproj-aqua-py

A python wrapper to provide a pip-installable
[aquaproj/aqua](https://github.com/aquaproj/aqua) binary.

Internally this package provides a convenient way to download the pre-built
aquaproj/aqua binary for your particular platform.

## installation

```bash
pip install aquaproj-aqua
```

## usage

After installation, the `aqua` binary should be available in your
environment (or `aqua.exe` on windows).
